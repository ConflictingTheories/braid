Tension Policies

How the Braid handles strands running at different speeds.

TightAtomic lock-step

Advances only when all strands yield. Slower strand holds the faster one until it catches up.

fiber-A
fiber-B
Output — synchronized tuples
Tuples: 0State: idle
const braid = new Braid([fiberA, fiberB]); // Tension.Tight is the default for await (const [a, b] of braid) { ... }
LooseFluid / cached

Fast strands use the last cached value of slow ones. Emits whenever any strand yields.

fast 120ms
slow 900ms
Output — fluid, slow value is cached
Fast: 0 Slow: 0 Emits: 0
const braid = new Braid([fast, slow], { tension: Tension.Loose });
FrayedStrict / fails fast

If any strand exhausts or throws, the entire Braid stops immediately. Strict pipelines only.

long (∞)
short (5)
Output — halts when short strand ends
Status: idle
const braid = new Braid([long, short], { tension: Tension.Frayed }); // stops when 'short' ends
Bind & ComposeFiber of Braids

braid.bind(fn) returns a new Fiber — which enters another Braid. Infinite nesting, no special API.

Level 1: [ask, bid] → spread
Level 2: [spread, signal] → decision
const spread = new Braid([ask, bid]) .bind(([a,b]) => a - b); // Fiber<number> const decision = new Braid([spread, sig]) .bind(([sp, s]) => sp > 2 ? s : "WAIT");

🧶 Braid

Weave async strands into synchronized cords

A language-agnostic co-routine framework for managing concurrent asynchronous flows.

$ yarn add @kderbyma/braid
Try Demos View on GitHub

Why Braid?

🔄

Lock-Step Synchronization

Coordinate multiple async streams with atomic precision.

🌊

Flexible Tension Modes

Tight, Loose, or Frayed modes for different concurrency needs.

🧩

Composable Design

Braids can become fibers, fibers can become braids.

⚡

Zero Dependencies

Lightweight with no external dependencies.

📦

Universal Imports

Works with ESM and CommonJS everywhere.

🎯

TypeScript Native

Full type safety with generics.

Live Demos

1. Tight Braid — lock-step

The Braid advances only when ALL strands have yielded a new value.

const letters = fromArray(["A", "B", "C"], "letters"); const numbers = fromArray([1, 2, 3], "numbers"); const braid = new Braid([letters, numbers]); const results = await Loom.collect(braid);
[0]["A",1]
[1]["B",2]
[2]["C",3]
2. Loose Braid — fluid / cached values

Fast strands use the last known value of slow strands. Emits whenever ANY strand produces.

const fast = new Fiber(async function*() { for (let i = 0; i < 5; i++) { yield `tick-${i}`; await sleep(20); } }, { name: "fast" }); const slow=new Fiber(async function*() { for (const h of ["🐂 Bullish", "🐻 Bearish" ]) { yield h; await sleep(80); } }, { name: "slow" }); const loose=new Braid([fast, slow], { tension: Tension.Loose });
[0]["tick-0","🐂 Bullish"]
[1]["tick-1","🐂 Bullish"]
[2]["tick-2","🐂 Bullish"]
[3]["tick-2","🐻 Bearish"]
[4]["tick-3","🐻 Bearish"]
[5]["tick-4","🐻 Bearish"]
3. Frayed Braid — stops on first exhaustion

If ANY strand exhausts before others, the entire Braid stops.

const short = fromArray([1, 2], "short"); // done after 2 const long = fromArray([10, 20, 30], "long"); // would continue const frayed = new Braid([short, long], { tension: Tension.Frayed });
[0][1,10]
[1][2,20]
4. bind() — composing a Braid into a Fiber

Transform braid output into a new Fiber.

const prices = fromArray([100, 101, 99, 103], "price"); const labels = fromArray(["AAPL", "AAPL", "AAPL", "AAPL"], "ticker"); const dashboard = new Braid([prices, labels]) .bind(([price, ticker]) => `${ticker}: $${price}`);
[0]"AAPL: $100"
[1]"AAPL: $101"
[2]"AAPL: $99"
[3]"AAPL: $103"
5. plait() — dynamic strand addition

Add a new Fiber to a running Braid.

const a = fromArray([1, 2, 3], "a"); const b = fromArray(["x", "y", "z"], "b"); const c = fromArray([true, false, true], "c"); const braid = new Braid([a, b]).plait(c);
[0][1,"x",true]
[1][2,"y",false]
[2][3,"z",true]
6. Fiber combinators — map / filter / take

Transform, filter, and limit streams with elegant combinators.

const nums = counter(0); // infinite: 0, 1, 2, 3 ... const processed = nums .filter((n) => n % 2 === 0) // only evens .map((n) => n * 10) // multiply .take(5); // stop after 5
[0]0
[1]20
[2]40
[3]60
[4]80
7. merge() — unordered fan-in

Merge N Fibers into one, emitting values as they arrive.

const a = new Fiber(async function*() { yield "a1"; await sleep(30); yield "a2"; }); const b = new Fiber(async function*() { await sleep(10); yield "b1"; await sleep(10); yield "b2"; }); const merged = merge(a, b);
[0]"a1"
[1]"b1"
[2]"b2"
[3]"a2"
8. Braid of Braids — deep composition

Compose braids into more complex structures.

const innerBraid = new Braid([ fromArray([1, 2, 3], "x"), fromArray([4, 5, 6], "y"), ]).bind(([x, y]) => x + y); const labels = fromArray(["sum1", "sum2", "sum3"], "label"); const outerBraid = new Braid([innerBraid, labels]) .bind(([sum, lbl]) => `${lbl} = ${sum}`);
[0]"sum1 = 5"
[1]"sum2 = 7"
[2]"sum3 = 9"
10. partition() — split by predicate

Split a Fiber into two based on a predicate.

const nums = fromArray([1, 2, 3, 4, 5, 6], "nums"); const [evens, odds] = partition(nums, (n) => n % 2 === 0); const braid = new Braid([evens.take(3), odds.take(3)]);
[0][2,1]
[1][4,3]
[2][6,5]
12. Real-world: stock ticker + news feed

Loose tension for real-time data with different update rates.

let price = 150; const ticker = new Fiber(async function*() { for (let i = 0; i < 6; i++) { yield price +=Math.round((Math.random() - 0.5) * 4); await sleep(15); } }, { name: "ticker" }); const news=new Fiber(async function*() { yield "📰 Earnings beat estimates" ; await sleep(50); yield "📰 Fed holds rates steady" ; }, { name: "news" }); const feed=new Braid([ticker, news], { tension: Tension.Loose }) .bind(([p, headline])=> `$${p} · ${headline}`);
