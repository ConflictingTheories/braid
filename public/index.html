🧶 Braid

Weave async strands into synchronized cords

A language-agnostic co-routine framework for managing concurrent asynchronous flows. Create elegant solutions for streams, real-time data, and parallel processing.

$ yarn add braid # or npm install braid
Try Demos View on GitHub

Why Braid?

🔄

Lock-Step Synchronization

Coordinate multiple async streams with atomic precision. Every value is perfectly aligned.

🌊

Flexible Tension Modes

Choose between Tight (synchronized), Loose (non-blocking), or Frayed (fault-tolerant) modes.

🧩

Composable Design

Braids can become fibers, fibers can become braids. Deep composition made simple.

⚡

Zero Dependencies

Lightweight with no external dependencies. Just pure TypeScript and async generators.

📦

Universal Imports

Works with ESM and CommonJS. Use it everywhere — Node.js, browsers, Deno.

🎯

TypeScript Native

Full type safety with generics. Your IDE knows exactly what's flowing through.

Use Cases

📊 Real-time Data Pipelines

Combine multiple data streams with different speeds. Handle stock tickers, sensor data, or user activity feeds while maintaining proper synchronization.

See Demo →
// Stock ticker + news feed const ticker = interval(price, 100); const news = interval(headline, 500); const feed = new Braid([ticker, news], { tension: Tension.Loose }).bind(([p, h]) => `$${p} · ${h}`);

🎮 Game State Management

Coordinate player inputs, game logic, and network sync. Create smooth multiplayer experiences with predictable state updates.

See Demo →
// Player inputs + game loop const keys = fromArray(keyEvents); const physics = counter(0, 16); const network = interval(serverState, 50); const gameState = new Braid([keys, physics, network]);

🌐 API Aggregation

Merge responses from multiple APIs with different latencies. Cache slow responses while processing fast ones.

See Demo →
// Multiple API calls const userApi = fetchUser(id); const postsApi = fetchPosts(id); const notifApi = fetchNotifs(id); const profile = new Braid([userApi, postsApi, notifApi]) .bind(([user, posts, notifs]) => ({ user, posts, notifs }));

🎵 Audio/Video Processing

Synchronize audio tracks, video frames, and metadata. Perfect for media applications requiring precise timing.

See Demo →
// Audio + video sync const audio = fromArray(audioFrames); const video = fromArray(videoFrames); const subtitles = fromArray(subtitles); const synced = new Braid([audio, video, subtitles]);

API Reference

Fiber

Wraps an AsyncGenerator and provides identity and combinator methods.

const fiber = new Fiber(gen, { name: 'myFiber' }); const mapped = fiber.map(x => x * 2); const filtered = fiber.filter(x => x > 10); const limited = fiber.take(5);

Braid

Weaves multiple Fibers together with configurable tension modes.

const braid = new Braid([a, b, c], { tension: Tension.Tight, timeout: 5000 });

Loom

Executor that runs Braids to completion with handlers.

await Loom.run(braid, (value, i) => { console.log(i, value); }); const results = await Loom.collect(braid);

Tension

Scheduling policy enum for Braid behavior.

Tension.Tight // Lock-step (default) Tension.Loose // Non-blocking Tension.Frayed // Fault-tolerant

Combinators

Functions for combining and transforming fibers.

merge(a, b, c) // Fan-in zip(a, b) // Binary zip chain(a, b) // Sequential partition(f, pred) // Split by predicate

Factories

Create fibers from common data sources.

fromArray([1,2,3]) counter(0, 100) // 0,1,2... every 100ms interval('ping', 1000) // Every second just(value) // Single value
