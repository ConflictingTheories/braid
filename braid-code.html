🧶 Braid: The "Boilerplate" Comparison

Both sides run the exact same async generators. Left manages them manually. Right uses your Braid framework.

⚠️ Native Async Generators
// 1. Manual Instantiation const iterA = genFast(); const iterB = genSlow(); try { while (true) { // 2. Manual Synchronization (Tight) const [r1, r2] = await Promise.all([ iterA.next(), iterB.next() ]); // 3. Manual Termination Check if (r1.done || r2.done) break; // 4. Value Extraction log(`[${r1.value}, ${r2.value}]`); } } catch (e) { log("Error: " + e); }
✅ Braid Framework
// 1. Wrap in Fiber const fibA = new Fiber(genFast); const fibB = new Fiber(genSlow); // 2. Create Braid (Handles Sync logic) const braid = new Braid([fibA, fibB], { tension: Tension.Tight }); // 3. Run Loom (Handles Termination/Errors) await Loom.run(braid, ([a, b]) => { log(`[${a}, ${b}]`); });
